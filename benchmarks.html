The AuspexAI Drift Benchmark

How much does an AI model's behavior actually change — over time, when settings change, or when the model itself is swapped? Measured on live volunteer hardware, with every number independently verifiable. Two views: one asks how far and how widely a model drifts; the other, how stable it is with fixed settings versus when sampling is turned on.

Drift is measured in multiples of the noise floor — the amount two identical setups naturally wobble run-to-run. 1.0 is that floor: anything below it is no real change; a value of 4 means the responses moved four times further than identical setups ever do on their own. Every measurement compares a model's responses on the same fixed prompts against one calibrated baseline run.

How to read these

Chart 1 — magnitude × breadth. Each dot is one model under one condition. Rightward = the single worst prompt moved further from baseline; upward = more of the prompts moved at all. The shaded strip on the left is the noise floor — dots inside it didn't really change. So the bottom-left corner is "stable," and the top-right is "drifts far and across the board." Colour marks whether the run was deterministic or used sampling.

Chart 2 — steady vs. sampled. One dot per model. The horizontal axis is how much it drifts with settings held fixed; the vertical axis is how much it drifts once sampling (temperature) is turned on. The diagonal is "equally sensitive to both"; a model above it is moved more by sampling than by anything else. This view needs both a fixed-settings run and a sampled run for each model, so it fills in as models are measured under both — a model measured only one way so far sits on its known axis with a hollow dot and a dashed “?” ray pointing to the axis still to be measured; it settles into the plane once it has been run both ways.

Why you can trust it

Every number is published by the researcher who ran the experiment, cryptographically signed, and anchored in a public transparency log — this page re-checks each signature in your browser, and the links in the table go to the public log records. Nothing here is hand-entered or editable: results are admitted automatically, by verification, not by an editor. Technical readers can re-derive every number from the underlying signed evidence — the raw registry and the open AuspexAI tooling make each figure independently reproducible.
